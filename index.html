BetterDex Instructions

Search Button D-Pad
This will search for the pokemon data after you put a name, or number that is a Pokemon. You can toggle through the next or previous Pokemon in the following way: Up-next pokemon Down-previous pokemon Left-back by 10 Right-forward by 10.
TCG Button Search Field
This will display card data for the current pokemon. You enter a valid number or name to search for pokemon here.
Red Power Button (top left) Yellow Instructions Button (top left)
Will power the BetterDex and open it, or power it off and close the BetterDex. This will allow you to see the text instructions again.
Poke-Data Button Evo. Button
This will display the pokemon data for the current pokemon. This will display the evolution forms of current pokemon.
Start Audio Button Stop Audio Button
This will play the instructions verbally while allowing the BetterDex to still be utilized. This will stop the audio instructions.
Quiz Buttons Map Button
This will open a quiz with varied difficulty, in a new window. This will pull up the maps of the different regions.
Left Arrow Button Right Arrow Button
This will go to the previous region map. This will go to the next region map.
Moves Button
This will display the moves of the current pokemon
